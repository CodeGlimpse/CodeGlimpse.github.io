这是一个简单高效的在线进制转换工具，支持二进制 (Base 2)、八进制 (Base 8)、十进制 (Base 10)、十六进制 (Base 16) 以及 2 到 36 之间的任意自定义进制之间的相互转换。

{{< tool id="binary" >}}

### 如何使用
1.  **选择源进制**：在下拉菜单中选择您要转换的数字的原始进制。
2.  **输入数值**：在“输入数值”框中输入您要转换的数字。
3.  **查看结果**：工具会自动实时显示转换为二进制、八进制、十进制、十六进制以及自定义进制后的结果。
4.  **自定义进制**：您可以通过修改自定义进制栏标题旁的数字来更改目标自定义进制。
5.  **复制结果**：点击每个结果右侧的“复制”按钮即可将其复制到剪贴板。

### 常见进制说明
-   **二进制 (Base 2)**：计算机内部使用的进制，只有 0 和 1 两个数字。
-   **八进制 (Base 8)**：由 0-7 八个数字组成，常用于早期计算机系统。
-   **十进制 (Base 10)**：人类通用的进制，由 0-9 十个数字组成。
-   **十六进制 (Base 16)**：由 0-9 和 A-F 组成，在编程、网页设计（如颜色代码）和内存地址中广泛使用。
